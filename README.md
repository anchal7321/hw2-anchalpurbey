# HW2 - Generative AI

## Business workflow
I chose a workflow that derives key pointers and actionable items  from the meeting notes

## Who the user is
The user is a project manager who needs to quickly translate insights from meetings to inform strategic direction for the project

## Input
The system receives AI transcribed meeting transcripts

## Output
The system produces 1) brief summary of important points to consider and 2)actionable next steps

## Why this is valuable
This task is valuable to save time, improve consistency, and reduce the effort needed after meetings.
